Rocket League is a game developed by Psyonix, and it was first released in July 2015. In September 2020, it was free-to-play. Rocket League is a game where eight players compete against each other using rocket-powered cars. Rocket League has a variety of multiplayer and single player modes that can be played both offline and online. It also added new game modes, such as ones based on ice hockey and hoops.
